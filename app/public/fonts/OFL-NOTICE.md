# Bundled fonts

Self-hosted so the app renders identically offline on desktop and web (no CDN).

- **Source Serif 4** — Copyright 2014–2023 Adobe (https://github.com/adobe-fonts/source-serif).
  Licensed under the SIL Open Font License 1.1. Used for all rendered tab text on
  screen (`SourceSerif4-*.woff2`) and embedded/subset into PDF exports
  (`SourceSerif4-*.woff`). Sourced via `@fontsource/source-serif-4`.
- **Noto Music** — Copyright The Noto Project Authors (https://github.com/notofonts/noto-fonts).
  Licensed under the SIL Open Font License 1.1. Embedded/subset into PDF exports for
  the musical rest glyphs and the tempo note (`NotoMusic.woff`). Sourced via
  `@fontsource/noto-music`.
- **Material Symbols Outlined** — Apache License 2.0 (UI icons).

The full SIL OFL 1.1 text is included with each font's npm package under
`node_modules/@fontsource/<name>/LICENSE`.
